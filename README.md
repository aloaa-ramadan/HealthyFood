🥗 Eat Healthy - Food App UI

📌 Overview

Eat Healthy is a mobile UI design created using Figma that helps users explore and choose healthy food بسهولة.

The design focuses on simplicity, clarity, and smooth user experience.

---

🎯 Features

- 🔐 Login & Sign Up screens
- 🔍 Search for food items
- 📂 Food categories (Soup, Fish, Meat)
- 🍽 view meals with prices and feedback 
- 📱 Clean and simple UI design

---

🧭 User Flow

1. User opens the app
2. Logs in or signs up
3. Browses categories
4. Searches for a meal
5. Views food items with prices

---

🎨 Screens Included

- Login Screen
- Sign Up Screen
- Home Screen
- Categories Section
- Meals List (Soup, Fish, Meat)
